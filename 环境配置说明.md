# 环境配置说明

本项目支持多环境配置，包括开发环境、测试环境和生产环境。

## 环境配置文件

- `.env` - 默认环境变量
- `.env.development` - 开发环境变量
- `.env.test` - 测试环境变量
- `.env.production` - 生产环境变量

## 环境变量说明

| 变量名 | 说明 | 示例 |
|--------|------|------|
| VITE_APP_TITLE | 应用标题 | 运营管理系统(测试) |
| VITE_APP_ENV | 环境标识 | test |
| VITE_API_BASE_URL | API基础地址 | https://privacy.test.biggerlens.com |
| VITE_APP_DEBUG | 调试模式 | true/false |

## 开发命令

### 启动开发服务器

```bash
# 开发环境（默认）
npm run dev

# 测试环境
npm run dev:test
```

### 构建项目

```bash
# 开发环境构建（默认）
npm run build

# 测试环境构建
npm run build:test

# 生产环境构建
npm run build:prod
```

构建完成后，文件将输出到 `dist` 目录：
- JS文件：`dist/js/`
- CSS文件：`dist/css/`

## 注意事项

1. 环境变量必须以 `VITE_` 开头才能在前端代码中访问
2. 修改环境变量后需要重启开发服务器
3. 不同环境的API地址会自动应用到开发服务器的代理配置中
4. 生产环境会自动移除console和debugger语句
